# Capture it like a pro

PWA móvil que coordina la grabación de parejas en sesiones de baile (bachata).
Prototipo de UI en un solo `index.html` autónomo — sin frameworks, sin build,
fuentes incrustadas.

**Demo:** https://avi-work-projects.github.io/capture-it-like-a-pro/

- Estilo "VISOR": la app vive dentro del visor de una cámara (timecode real,
  OSD, acento de color que cambia por pantalla).
- Pantallas: rol → filtros en cascada (país/ciudad/tipo/ambiente) → lista de
  eventos → detalle con camarógrafos y "apuntarme".
- Tema claro/oscuro con persistencia. Datos mock para probar (España y Polonia).

Más contexto del producto en [PROYECTO.md](PROYECTO.md). Variantes de estilo
exploradas en [styles/](styles/).
